# MusicSorter - 音樂自動分類與整理工具

MusicSorter 是一個基於 Python 開發的輕量級自動化工具，旨在幫助音樂愛好者快速解決雜亂的音樂檔案整理問題。只需一鍵，即可根據檔案資訊自動分類並歸檔您的音樂庫。

## 主要功能
* **智慧分類**：自動掃描指定目錄下的音樂檔案，並按照設定規則進行分類整理。
* **高效輕量**：採用 CPU 原生運算，不佔用系統 GPU 資源，不駐留背景，對電腦效能零負擔。
* **簡單易用**：無需安裝複雜依賴環境，直接下載 `MusicSorter.exe` 即可在 Windows 上運行。
* **開源透明**：所有原始碼皆公開，方便您自行審查與優化。

## 掃描效能說明
* **核心運作**：本程式掃描與分類過程皆由 **CPU** 執行。由於程式邏輯主要為檔案 I/O 操作（讀取路徑、解析標籤、移動檔案），並非複雜的數學計算，因此無需 GPU 加速。
* **影響速度的因素**：
  **硬碟類型**：使用 SSD（固態硬碟）的速度會遠快於傳統 HDD（機械硬碟），因為掃描檔案目錄的速度取決於硬碟的讀取 IOPS。
  **檔案數量**：音樂庫中的檔案總數越多，掃描所需的時間會成線性增加。
  **檔案標籤 (Metadata)**：若音樂檔案包含大量且複雜的 ID3 Tag，CPU 解析標籤的時間也會略微影響整體速度。

## 支援系統
* **Windows 10 / 11**：本軟體已在 Windows 環境下進行測試並打包為獨立執行檔 (.exe)。
* **跨平台需求**：如需在 macOS 或 Linux 上運行，建議您直接使用 `safemusic.py` 源碼執行，確保環境已安裝 Python 3.x。

## 使用說明

1. 前往 [Releases 頁面](https://github.com/FTTBEE/MusicSorter/releases) 下載最新版的 `MusicSorter.exe`。
2. 將該執行檔放入您的音樂檔案夾內。
3. 雙擊執行程式，它將自動掃描並建立分類目錄。
4. 整理完成後，程式會顯示處理結果，您可以隨時手動關閉視窗。
