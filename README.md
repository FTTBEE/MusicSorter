# MusicSorter

這是一個基於 Python 的音樂自動分類與播放工具。它可以幫助您快速整理雜亂的音樂檔案，將它們按照預設規則進行分類。

## 功能特點
* **自動分類**：將混亂的音樂檔案自動分類到對應的資料夾。
* **輕量化**：無需複雜安裝，下載 .exe 即可運行。

## 如何使用
1. 在 [Releases](https://github.com/FTTBEE/MusicSorter/releases) 頁面下載最新版的 `MusicSorter.exe`。
2. 將程式放置在包含音樂檔案的資料夾中。
3. 執行程式，它將自動幫您整理音樂。
